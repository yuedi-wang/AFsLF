# AFsLF
Official implementation of **AFsLF (Autofocus for Scanning Light Field Microscopy)**, a real-time robust autofocus method for sustained intravital light field imaging.

---

## Overview
This repository contains the core algorithm code of AFsLF, which enables long-term stable high-resolution intravital imaging by real-time defocus estimation and focal plane correction for scanning light field microscopy (sLFM) systems.
